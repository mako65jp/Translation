Extension Anatomy
In the last topic, you were able to get a basic extension running. How does it work under the hood?

The Hello World extension does 3 things:

Registers the onCommand Activation Event: onCommand:extension.helloWorld, so the extension becomes
activated when user runs the Hello World command.
Uses the contributes.commands Contribution Point to make the command Hello World available in the
Command Palette, and bind it to a command ID extension.helloWorld.
Uses the commands.registerCommand VS Code API to bind a function to the registered command ID
extension.helloWorld.
